# RACE — Snooker Scoreboard

A single-file, offline-friendly snooker scoring app. Dark felt-green & gold, tactile ball buttons, live leaderboard, race-to-target gameplay, and a hidden visitor log.

## Use it
Open `index.html` in any browser — no build, no dependencies, works offline.

## Features
- Setup: any number of players + a target score ("race to")
- Ball scoring (Red 10, Black 7, Pink 6, Blue 5, Brown 4, Green 3, Yellow 2) and fouls (−10 on red, −4 on others)
- Fixed-position consoles and leaderboard; per-player undo
- Race continues until the last player — each finisher gets a ranked announcement, then a final results screen
- First-visit name prompt logged to `localStorage`; hidden admin panel (triple-tap the dot, top-right, or Ctrl/Cmd+Shift+A)
